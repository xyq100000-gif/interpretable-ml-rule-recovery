# CV Project Entry Options

## Option A — concise CV bullet set

**From Classification to Rule Recovery: An Interpretable ML Pipeline**  
Refactored a machine learning coursework submission into a research-oriented case study on interpretable structure discovery from synthetic tabular data. Designed a three-stage pipeline for multiclass classification, within-group ordinal rule recovery, and sparse linear formula reconstruction using feature screening, repeated cross-validation, threshold search, and constrained model selection. Recovered a 2-feature integer formula for a latent target with training R² = 0.9993.

- Implemented Fisher-score feature screening, hand-coded KNN with repeated stratified cross-validation, per-group threshold optimisation, and sparse OLS subset search; recovered the formula `prowess = -27796 + 6·charring + 887·sliminess` with R² = 0.9993.
